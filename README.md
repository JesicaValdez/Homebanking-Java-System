**Proyecto Integrador Final | Java EE (Servlets) & MySQL**

Este sistema de banca digital fue desarrollado como un **trabajo colaborativo en equipo (6 integrantes)** para el proyecto final de la carrera. Simula un entorno bancario real, con un fuerte enfoque en la integridad de los datos, transacciones seguras y control administrativo.

### 👥 Colaboración y Rol
El proyecto se desarrolló bajo una metodología de alto rendimiento donde equilibré los roles de **Desarrollador Backend** y **Project Manager**, asegurando el cumplimiento de los plazos de entrega (sprints).

### 🛠️ Stack Tecnológico
* **Lenguaje:** Java (Principios de POO).
* **Tecnología Web:** **Java Servlets & JSP** (Manejo de peticiones HTTP, gestión de sesiones y contenido dinámico).
* **Base de Datos:** MySQL (Esquema relacional y Procedimientos Almacenados).
* **Conectividad:** JDBC para una comunicación segura con la base de datos.
* **Servidor:** Apache Tomcat.

### 🚀 Funcionalidades Principales (ABM / CRUD)
El sistema cuenta con un panel administrativo completo y una interfaz para clientes:

* **Gestión de Clientes y Usuarios:** ABM completo de clientes bancarios y usuarios del sistema, incluyendo autenticación segura y gestión de perfiles mediante Servlets.
* **Administración de Cuentas:** Gestión de diferentes tipos de cuentas (Ahorro/Corriente), con generación automática de números de cuenta y seguimiento de estado.
* **Sistema de Préstamos:** ABM de préstamos con cálculo de tasas de interés, cronogramas de pago y flujos de aprobación.
* **Motor de Transacciones (Movimientos):** Procesamiento en tiempo real de transferencias, depósitos y retiros con registro completo de historial para auditoría.
* **Reportes:** Generación dinámica de reportes financieros y estados de cuenta.

### 🔧 Diseño de Base de Datos
El proyecto utiliza un backend en **MySQL** con un esquema normalizado para garantizar las propiedades **ACID** en cada transacción, evitando la pérdida o duplicidad de datos durante operaciones concurrentes.
